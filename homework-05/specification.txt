Перед выполнением домашки вам нужно скопировать файлы из базового проекта в свой проект (внизу ссылка на github).

Коротко:
Прочитать все запакованные JSON файлы и преобразовать их в один, общий JSON файл по шаблону.
Итоговый JSON файл должен быть запакованн.
Файлы для преобразования находятся в корне домашки в папке ./input/
Результат нужно сохранить в ./output/result.json.gz
Как преобразовать файлы показано внизу на картинке.

Подробно:
Нужно получить список файлов и преобразовать каждый файл в JS объект используя модули fs, path, zlib(gunzip и gzip)

В каждом полученном объекте есть поле url которое содержит ссылку. Эту ссылку нужно преобразовать с помощью модуля URL по шаблону:
- protocol: http: заментить на https:
- search params: на базе названия файла из pathname собрать запрос (query) ?file=filename&type=.fileextension
( напр. из названия файла keyboards.xls должно получится ?file=keyboards&type=.xls )
- pathname: заменить на /devices

Каждый объект из файла нужно добавить в общий объект с имнем. Имя считываемого JSON файла будет именем поля.
 ( напр. название JSON файла monitor.json.gz преобразовать в строку monitor и использовать эту строку в качестве имени поля, а объект этого файла использовать как значение поля. )

Общий объект нужно превратить в запакованный JSON и сохранить по пути ./output/result.json.gz
